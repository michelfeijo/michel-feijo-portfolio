# Michel Feijó — Portfolio

🔗 **[Acesse o site](https://michelfeijo.github.io/michel-feijo-portfolio/)**

Portfólio pessoal desenvolvido para apresentar minha trajetória como
Desenvolvedor Full Stack, minhas tecnologias e projetos em destaque.

## 🛠️ Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Radix UI

## 🚀 Rodando localmente

```bash
pnpm install
pnpm --filter @workspace/portfolio dev
```

## 📦 Deploy

Deploy automático via GitHub Actions a cada push na branch `main`,
publicado no GitHub Pages.
